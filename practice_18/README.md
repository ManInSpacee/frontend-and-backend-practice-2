# Фронтенд и бэкенд разработка — практики 13–17

## Практика 13–14. PWA + Service Worker
Реализовано одностраничное TODO-приложение с поддержкой Progressive Web App: задачи хранятся в `localStorage`, интерфейс адаптирован под мобильные устройства. Подключён Service Worker со стратегией Cache First — приложение работает офлайн после первой загрузки. Добавлен `manifest.json` с иконками (генерируются скриптом через Canvas), что позволяет установить приложение на домашний экран.

## Практика 15. HTTPS + App Shell
Выполнено:
- Настроен локальный HTTPS через `mkcert` (сертификаты `localhost+2.pem`).
- Запуск через `http-server --ssl` на порту 3000.
- Реализована архитектура App Shell: каркас (`index.html`) кэшируется при установке SW, контент страниц (`pages/home.html`, `pages/about.html`) подгружается динамически через `fetch`.
- Service Worker: статика — Cache First, динамические страницы — Network First с фолбеком из кэша.
- Генерация иконок через `generate-icons.js` (canvas).

## Практика 16. WebSocket + Push
Выполнено:
- Node.js сервер на Express + HTTPS, порт **3001**.
- Подключён `socket.io` для двусторонней связи в реальном времени.
- Сгенерированы VAPID-ключи (хранятся в `vapid.json`), настроен `web-push`.
- Эндпоинты `/subscribe` и `/unsubscribe` для управления push-подписками.
- На клиенте: кнопки «Включить / Отключить уведомления», функции `subscribeToPush` / `unsubscribeFromPush` через `PushManager`.
- При добавлении задачи отправляется событие `newTask` → сервер рассылает `taskAdded` всем клиентам и шлёт push-уведомление подписчикам.
- Service Worker обрабатывает событие `push` и показывает системное уведомление.

## Практика 17. Детализация Push (напоминания)
Выполнено:
- Сервер на Express + socket.io, порт **3002**.
- Добавлена форма создания заметки с напоминанием (текст + `datetime-local`).
- Структура заметки в `localStorage` расширена: `id`, `text`, `reminder` (timestamp).
- На сервере планирование push через `setTimeout`, активные таймеры хранятся в `Map`.
- Событие `newReminder` от клиента → сервер ставит таймер → в нужное время шлёт push с `reminderId`.
- Эндпоинт `/snooze` — переносит напоминание на 5 минут.
- Service Worker: в `push`-обработчике добавлена кнопка действия «Отложить на 5 минут», в `notificationclick` — обработка действия `snooze` с запросом на сервер.
- Добавлена страница с React-компонентом `UsersPage` (`src/pages/UsersPage.jsx` + `.scss`).
